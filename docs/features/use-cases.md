## For Buyers (Using Ragora)

### Customer Support Bot

Build a support agent that answers questions using your documentation.

**Setup:**
1. Create a workspace for support docs
2. Upload product manuals, FAQs, troubleshooting guides
3. Connect via API or integrate with Discord/Slack

**Why Ragora?**
- ✅ Answers grounded in actual documentation
- ✅ Auto-updates when docs change
- ✅ Cites sources for verification

---

### Internal Workspace

Make company knowledge searchable by AI assistants.

| Source Type | Examples |
|-------------|----------|
| HR Policies | PTO, expenses, benefits |
| Engineering | Architecture docs, runbooks |
| Sales | Pitch decks, case studies |
| Legal | Contracts, compliance docs |

---

### Developer Documentation Search

Make code documentation and APIs searchable.

**What to Index:**
- README files and technical specs
- API documentation
- Architecture decision records
- Runbooks and playbooks

**Setup:**
1. Connect your GitHub repositories
2. Filter to include `docs/`, `*.md`
3. Use in your IDE with MCP

---

### Compliance & Legal Assistant

Build AI that cites exact policy clauses.

**Example:** _"Can we approve this vendor without SOC2 certification?"_

**Without Ragora:** Vague "it depends" answer.

**With Ragora:** "No. According to Section 4.2 of your Vendor Risk Policy, all vendors must be SOC2 Type II compliant." _(with citation)_

Search across papers, reports, and research documents.

## For Sellers (Monetizing Knowledge)

Ragora enables **knowledge monetization** — earn revenue when AI agents access your content.

### Discord Community Monetization

**The Pitch:** _"You have 5,000 members and 50 experts answering questions for free. Turn solved threads into passive income."_

**How It Works:**
1. Install the Ragora Discord bot
2. Bot indexes threads marked "Solved"
3. When AI agents query your community knowledge, you earn revenue.

**Best For:**
- Tech support communities
- Framework-specific Discords
- Gaming and crypto communities

---

### API/Library Documentation

**The Pitch:** _"ChatGPT is trained on your v1.0 docs, but you're on v3.0. Developers spend more time because AI writes broken code."_

**How It Works:**
1. Connect your GitHub docs repo via Data Sources
2. Recurring sync keeps docs up-to-date
3. AI agents get real-time, accurate documentation
4. Reduce "this doesn't work!" GitHub issues

**Best For:**
- private repos
- Open source maintainers
- SDK/API providers
- Fast-moving libraries (LangChain, Next.js, etc.)

---

### Influencer Workspace

**The Pitch:** _"You have 100k followers and 500 DMs a day asking for advice. Clone your brain."_

**How It Works:**
1. Upload video transcripts, newsletters, tweets
2. Developers build apps that "Chat with You"
3. Earn royalties on every question answered
4. Inject affiliate links into responses

**Best For:**
- Tech YouTubers
- Course creators
- Thought leaders

---

### Product Specifications

**The Pitch:** _"Users ask 'Does this camera fit a 50mm lens?' but the answer is buried in a PDF manual."_

**How It Works:**
1. Upload product manuals and spec sheets
2. Shopping agents query structured specs
3. Earn revenue when specs are retrieved

**Best For:**
- Manufacturers
- E-commerce support
- Technical product companies
