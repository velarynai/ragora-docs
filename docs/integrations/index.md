# Integrations

Ragora integrates with the tools you already use — AI assistants, websites, data sources, and autonomous agents. Each integration lets you search, retrieve, or sync data with your Ragora workspaces in a different way.

## Available Integrations

| Integration | What It Does | Status |
|-------------|-------------|--------|
| [**MCP**](./mcp-guide) | Connect AI assistants like Claude Desktop, Cursor, and VS Code to search your workspaces via the Model Context Protocol | Available |
| [**Cloud Connectors**](./connectors) | Recurring sync from external data sources (websites, GitHub, Google Drive, etc.) into workspaces | Docs Website available; others coming soon |
| [**OpenClaw**](./openclaw) | Give your self-hosted autonomous AI agent access to Ragora workspaces | Available |
| [**Chat Widget**](./chat-widget) | Embed an AI chatbot on any website with a single script tag | Available |

## Choosing an Integration

- **Want your AI coding assistant to search your docs?** → [MCP Integration](./mcp-guide)
- **Want to keep a workspace in sync with an external source?** → [Cloud Connectors](./connectors)
- **Want to add AI chat to your website?** → [Chat Widget](./chat-widget)
- **Want a self-hosted agent with Ragora access?** → [OpenClaw](./openclaw)
- **Want bots in Discord or Slack?** → See [Bots](/docs/bots)
